🎯 Hook Variants — 3 per Kit

Every kit's script gets three openers, one per category. Run them as A/B/C tests and let the algorithm tell you which works in each market. Click any script to copy. Use file-naming convention from Step 10: Kit[N]_[Category]_[VariantID]_v[N].mp4

The 3 categories:
  • 🌀 Curiosity — opens with a question or symptom that creates a "wait, that's me" pause. Best for cold audiences.
  • ⚡ Contradiction — counter-narrative ("it's not X, it's Y"). Best for audiences already self-diagnosing wrong.
  • 🎯 Direct-Result — opens with the outcome already achieved ("in 7 days, my X stopped"). Best for retargeting / proof.
Rotation rule: Post one of each per week per kit. Track which category wins in each market — they will not all win in the same place. Indian audiences often prefer authority/contradiction; Brazilian audiences often prefer direct-result.

Doubling rule

If one variant outperforms the others by 3× views or 5× comments for two weeks running in a market: produce two of that category next week and drop the worst-performing category. Run the experiment for 4 weeks before declaring a market preference.
