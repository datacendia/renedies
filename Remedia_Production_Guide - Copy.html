Production Steps (Per Video)

Replace [KIT] with your market number. Do all 9 steps for Kit 1 (Global) first, then repeat for Kits 2–5.

🟢 Big-picture flow: Train your character first (Steps 1–2) → Generate hero image with that character (Step 3) → Lipsync it (Step 4) → Add cinematic b-rolls (Steps 5–7) → Edit (Step 8) → Publish (Step 9).

Step 1 — Generate Soul ID Training Set

⏱ 30 min · Tool: Nano Banana Pro

Goal: 20–30 high-quality images of the same character to feed into Soul ID. Single-image training = "Bad" results. Volume + consistency = "Excellent".

  1. Sidebar → Image → Nano Banana Pro (or Ctrl+K → "Nano Banana")
  2. Open Soul ID Training Prompts and pick 8–10 prompts from your kit's set:
    • Kit 1 → T1.1 through T1.20
    • Kit 2 → T2.1 through T2.20
    • Kit 3 → T3.1 through T3.20
    • Kit 4 → T4.1 through T4.20
    • Kit 5 → T5.1 through T5.20
  3. For each prompt: Aspect 9:16, Quality High, Variations 4 → Generate
  4. Reject any image where face shape, eye color, hair texture, or age drifts. Keep only "clearly the same person" results.
  5. Target 20–30 keepers minimum. Save them all to [Market]/training/
Quality gate: Don't proceed to Step 2 until you have 20+ keepers that match the locked Character Bible for your kit. Garbage in = garbage Soul ID.

Step 2 — Train Soul ID Character

⏱ 10 min upload + 5–10 min training · Tool: Soul ID
  1. Sidebar → Image → Soul ID
  2. Click Create New Character, name it Remedia_[Market] (e.g. Remedia_Global)
  3. Drag in all 20–30 training images from [Market]/training/ at once
  4. Watch the quality meter. Add or remove images until it shows "Good" or "Excellent"
  5. Click Create ✨ 25 (this is the "train" button; ✨ 25 = 25 Higgsfield credits)
  6. Wait 5–10 min until the character card shows "Ready"

Step 3 — Generate Hero Herbalist Image (with Soul ID)

Step 8 — Assemble in CapCut

⏱ 15 min · Tool: CapCut Desktop
  1. New project, ratio 9:16
  2. Import all clips into media library
  3. Build the timeline (see below)
  4. Detach the talking_head audio so VO continues through B-roll cutaways
  5. Auto-Captions: Captions → Auto Captions → manually set Source Language (don't trust auto-detect):
    • Kit 1 → English · Kit 2 → Hindi · Kit 3 → Spanish · Kit 4 → Chinese (Simplified) · Kit 5 → Japanese
  6. Caption font (critical): Set font to Noto Sans (or Noto Sans CJK for Kits 4/5, Noto Sans Devanagari for Kit 2). Noto covers every script — most other fonts will render non-Latin scripts as garbled boxes/glyphs.
  7. Caption style: 80–90px, white text, black stroke 4px, center-aligned, positioned in the bottom 25% safe area (above the mobile UI zone)
  8. Music: trending ambient/world at -22dB on Track 3
  9. Color: Vintage Warm filter at 60%
  10. Export 1080×1920, 30fps, High bitrate, MP4 H.264
