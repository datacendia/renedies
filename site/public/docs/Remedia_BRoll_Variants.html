🎬 B-Roll Variants — 3 Alternates per Kit

Each kit's existing 3 b-rolls (X.2–X.4 in the Production Guide) get 3 alternative b-rolls so videos don't look identical when cross-posted. Algorithms penalize duplicate visual content. Click any prompt to copy.

The 3 alternative styles (one per kit slot):
  • ✋ Close-up Hands — replaces the powder/pour b-roll. The herbalist's hands working a specific gesture (grinding, sifting, tying).
  • 🌾 Ingredient Pour — replaces the tea/cup b-roll. Pure ingredient action, no human present.
  • 💨 Steam Swirl — replaces the hanging-herbs b-roll. Hyper-detailed steam/vapor over the prepared remedy.
Cross-post mapping:
  • Original cut → TikTok / Instagram Reels (uses X.2, X.3, X.4)
  • Variant cut A → YouTube Shorts (uses X.2-ALT, X.3, X.4)
  • Variant cut B → Pinterest carousel + Lemon8 (uses X.2, X.3-ALT, X.4-ALT)
Different platforms get different b-rolls. Same talking-head MP4. The visual mix reads as fresh content.

📋 How to Use the Variants

Step 1 — Generate the alternates (one-time)

  1. For each kit you've already produced the original 3 b-rolls for, generate the matching 3 alternates from this page
  2. Use Nano Banana Pro for stills, then Higgsfield Create Video for motion (same as original workflow)
  3. Save into [Market]/broll/alt/ alongside [Market]/broll/

Step 2 — Build platform-specific cuts

PlatformB-Roll MixWhy
TikTok / ReelsX.2 + X.3 + X.4 (originals)First post — algorithm sees fresh content.
YouTube ShortsX.2-ALT + X.3 + X.4Shorts algorithm checks visual signature first 2s. Different opener = treated as fresh.
Pinterest carouselX.2 + X.3-ALT + X.4-ALTPinterest's visual search needs distinct stills — alternates differentiate.
Lemon8 / Red NoteX.2-ALT + X.3-ALT + X.4Heavy aesthetics-driven feeds. Original may have been seen by users who cross-platform.
WhatsApp / Telegram ChannelsAll ALT (X.2-ALT + X.3-ALT + X.4-ALT)Subscribers want something they haven't seen on public feeds.

Step 3 — Track which mix wins

Add a _mix[A|B|C] tag to filenames. After 30 days, identify which mix outperforms in each market. Originals are not always the winners — the close-up-hands variant often beats the powder-pour for thumb-stopping power on Reels.
